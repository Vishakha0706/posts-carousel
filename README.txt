== Description ==

This plugin creates shortcode for display posts carousel with some parameters.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `posts-carousel` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
